# User Testing Instructions & Feedback Collection

## Welcome to the Order Management System Test!

Thank you for helping us test our order management system. Your feedback is crucial for improving the user experience and ensuring the system works as expected.

## Test Session Overview

- **Duration**: 45-60 minutes
- **Environment**: Test system (not production)
- **Goal**: Validate business workflows and user experience
- **Your Role**: [Seller/Customer/Admin] (assigned below)

## Before You Begin

### System Requirements
- Computer with internet connection
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Test account credentials (provided below)

### Your Test Assignment
**Please check your assigned role:**

- **Seller Test**: Focus on product management, order processing, customer management
- **Customer Test**: Focus on browsing, ordering, and tracking
- **Admin Test**: Focus on user management, system monitoring, support

## Step-by-Step Test Instructions

### Phase 1: Setup & Login (5 minutes)

1. **Open Test Environment**
   - URL: `http://localhost:3000`
   - Bookmark this page for easy access

2. **Login with Your Credentials**
   ```
   [INSERT YOUR ASSIGNED CREDENTIALS HERE]
   Email: [YOUR_EMAIL]
   Password: [YOUR_PASSWORD]
   ```

3. **Familiarize Yourself**
   - Look around the interface
   - Note your first impressions
   - Don't worry about making mistakes - this is a test environment!

### Phase 2: Core Task Completion (30-40 minutes)

**Complete the scenarios assigned to your role:**

#### For SELLER Users:
1. **Product Management** (10 minutes)
   - Create a new product
   - Update an existing product
   - Try deactivating a product

2. **Order Processing** (15 minutes)
   - View incoming orders
   - Process a pending order
   - Update order status through the workflow
   - Try cancelling an order

3. **Customer Management** (5 minutes)
   - View customer information
   - Check order history

#### For CUSTOMER Users:
1. **Browse & Order** (15 minutes)
   - Browse available products
   - Place a single-item order
   - Place a multi-item order
   - Try ordering with custom notes

2. **Order Tracking** (10 minutes)
   - Check order status
   - View order details
   - Try finding past orders

#### For ADMIN Users:
1. **User Management** (15 minutes)
   - View user lists
   - Try creating a new seller account
   - Test user permission changes

2. **System Monitoring** (15 minutes)
   - Check dashboard metrics
   - Review system logs
   - Handle a support scenario

### Phase 3: Exploration & Error Testing (10 minutes)

1. **Try to Break Things**
   - Submit invalid data
   - Test edge cases
   - Try confusing workflows

2. **Note Any Issues**
   - What was confusing?
   - What didn't work as expected?
   - What could be better?

### Phase 4: Feedback Collection (5 minutes)

Complete the feedback questionnaire below.

## Important Testing Guidelines

### DO:
- Think aloud while performing tasks
- Ask questions if anything is unclear
- Report all issues, even minor ones
- Try realistic workflows
- Take your time - accuracy over speed

### DON'T:
- Worry about breaking the system
- Skip tasks because they seem difficult
- Assume something is "user error" - report it anyway
- Use your real personal information
