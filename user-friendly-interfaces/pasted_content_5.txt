## 3. ❌ Missing State Machine Component

You mentioned:

> “Live State Machine Visualizer”

But no implementation.

---

## ✅ Minimal Real Component (not design fluff)

```tsx
type Props = {
  current: OrderStatus;
  onChange: (next: OrderStatus) => void;
};

export function StateMachine({ current, onChange }: Props) {
  const allowed = getAllowedTransitions(current);

  return (
    <div className="flex gap-2">
      {Object.keys(ORDER_TRANSITIONS).map((state) => {
        const isActive = state === current;
        const isAllowed = allowed.includes(state as OrderStatus);

        return (
          <button
            key={state}
            disabled={!isAllowed}
            onClick={() => onChange(state as OrderStatus)}
            className={`
              px-3 py-2 rounded
              ${isActive ? "bg-indigo-600 text-white" : "bg-gray-100"}
              ${!isAllowed ? "opacity-30 cursor-not-allowed" : ""}
            `}
          >
            {state}
          </button>
        );
      })}
    </div>
  );
}
```

👉 This enforces:

* state visibility
* transition constraints
* interaction control
