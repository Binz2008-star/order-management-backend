---
description: Implement a feature — service first, route second, test third
---

1. Understand the feature request.
2. Implement the service layer logic first in `src/server/services/`.
3. Add the thin route handler in `src/app/api/`.
4. Add tests using factories from `src/tests/factories/`.
5. Run typecheck and tests to verify.
6. Report: files created/changed, key decisions, what to test.
